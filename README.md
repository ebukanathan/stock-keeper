Express Session RBAC Auth

A TypeScript + Express.js authentication system with session-based login and role-based access control (RBAC). Secure, type-safe, and ready for multi-role applications like admin dashboards, user portals, or any system with restricted routes.

Features

Session-based authentication using express-session

 Role-based access control (RBAC) middleware

 Type-safe session extensions (userId & role)

 Secure cookies (httpOnly, sameSite, secure)

 Easily extensible roles (admin, user, etc.)

 Ready-to-use middleware for protecting routes
